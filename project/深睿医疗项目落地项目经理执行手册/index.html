一、项目核心目标（PM 必明确）

  1. 按期完成深睿 MetAI/TrioData X 与院内系统对接，无业务中断

  2. 满足临床科室核心需求（影像 AI 辅助诊断、报告结构化、科研数据支撑）

  3. 通过合规验收（三类证核查、数据安全等保、流程留痕）

  4. 控制成本在预算内，算力 / 存储资源利用率≥80%

四、IT 对接核心管控点（PM 重点盯防）

  1. 数据对接：

  • 要求深睿提供《接口对接规范手册》，明确 DICOM 标签映射、HL7 消息格式

  • 关键校验：影像传输完整性（无丢帧）、报告回写字段准确性（与 EMR 字段一致）

  1. 部署架构：

  • 本地部署：确认 GPU 服务器配置（推荐≥2 张 A100，显存≥40GB）、存储扩容方案（预留≥10TB 影像缓存）

  • 网络要求：院内局域网带宽≥10G，深睿系统与 PACS/RIS 网络隔离（避免攻击风险）

  1. 合规管控：

  • 收集深睿 14 张三类证复印件、医疗器械注册证附件（加盖公章）

  • 签署《数据安全保密协议》，明确数据所有权、使用范围（避免合规审计风险）

五、预算与成本控制

成本项 控制要点
硬件采购（GPU / 存储） 优先利旧医院现有 GPU，如需采购 → 对比 3 家供应商报价

培训与驻场费 整合培训场次（减少深睿工程师驻场天数）
