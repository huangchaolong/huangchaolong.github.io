##医共体数据中心应用服务启动文档
##服务器IP地址：192.168.21.13

注册中心，第一个启动:

nohup java -jar /home/servers/local/agent/XFoundationXRegistrationAgent-4.0.0.jar  >  /home/servers/local/agent/log.txt &

网关服务:

nohup java -jar /home/servers/local/gateway/XDataIntelligenceXGateway-1.0.0.jar  >  /home/servers/local/gateway/log.txt &

用户管理:

nohup java -jar /home/servers/local/userManage/XDataIntelligenceXUserManagement-1.0.0.jar >  /home/servers/local/userManage/log.txt &

登录模块:

nohup java -jar /home/servers/local/base/XDataIntelligenceXBase-1.0.0.jar >  /home/servers/local/base/log.txt &

全息病历:

nohup java -jar /home/servers/local/clinical/XDataIntelligenceXClinical-1.0.0.jar >  /home/servers/local/clinical/log.txt &

主数据管理:

nohup java -jar /home/servers/local/mainData/XDataIntelligenceXMainData-1.0.0.jar >  /home/servers/local/mainData/log.txt &

大屏模块:

nohup java -jar /home/servers/local/bigScreen/XDataIntelligenceXBigScreen-0.0.1-SNAPSHOT.jar >  /home/servers/local/bigScreen/log.txt &

双向转诊:

nohup java -jar /home/servers/local/dualReferral/XDataIntelligenceXDualReferral-1.0.0.jar >  /home/servers/local/dualReferral/log.txt &

数据对外服务:

nohup java -jar /home/servers/local/dataServices/XDataIntelligenceXDataService-1.0.0.jar >  /home/servers/local/dataServices/log.txt &

自动执行任务模块(两个按顺序执行):

nohup java -jar /home/servers/local/job-admin/xxl-job-admin-2.2.1-SNAPSHOT.jar >  /home/servers/local/job-admin/log.txt &
nohup java -jar /home/servers/local/job-exec/xxl-job-executor-sample-springboot-2.2.0.jar >  /home/servers/local/job-exec/log.txt &

tomcat：

cd /home/servers/local/apache-tomcat-8.5.76/bin/
sh startup.sh

nginx:

cd /usr/local/nginx/sbin
./nginx

停止操作-jar包:

ps -ef|grep java
kill -9 进程号(进程号为上一个命令查出的第二列的数字,需要关掉哪个模块，就输入哪个的进程号,模块根据最后一列jar包模块确认)

停止操作-tomcat：：

cd /home/servers/local/apache-tomcat-8.5.76/bin/
sh shutdown.sh

停止操作-nginx:：

cd /usr/local/nginx/sbin
./nginx -s stop

部署前端(启动时不用执行,部署前端包时执行)：
解压：unzip -o

/home/servers/local/dist/5.18-1.zip -d /home/servers/local/dist/xdata-intelligence-view

复制：\cp -rf

/home/servers/local/dist/xdata-intelligence-view/dist/. /home/servers/local/dist/xdata-intelligence-view
